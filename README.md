# -BCD-to-Excess3-Converter-

A digital logic project designed and implemented by Team OctaCore to convert BCD (Binary Coded Decimal) input into its corresponding Excess-3 code output.

📌 Project Overview

The BCD to Excess-3 Converter is a combinational logic circuit that converts a 4-bit BCD input into a 4-bit Excess-3 output.

Excess-3 is generated by adding 0011 to the BCD input. This code is commonly used in digital electronics and code conversion applications.

🎯 Objective
Design a combinational circuit to convert BCD to Excess-3.
Build the truth table of the converter.
Simplify Boolean equations using Karnaugh Maps (K-Map).
Implement the circuit using logic gates.
Test and verify the output practically.

 Testing and Output Display

The circuit was tested practically by applying the BCD input using LEDs and a Seven-Segment Display.

Input:
4 LEDs were used to indicate the BCD binary input.
A Seven-Segment Display was connected to show the decimal digit corresponding to the input.
Output:
4 LEDs were used to display the Excess-3 output bits.

Each input from 0 to 9 was tested, and the output matched the expected Excess-3 code successfully.
